# Cyclotourix

Ce dépôt contient le code source du site [cyclotourix.fr](https://cyclotourix.fr/),
vitrine d'un événement cyclotouriste dont la première étape est prévue pour
septembre 2026.

## L'événement

La **Cyclotourix** est un voyage à vélo en itinérance, gratuit et ouvert à tous.

L'idée : parcourir en groupe des routes côtières et bocagères, se retrouver le
matin et le soir, et découvrir les spécialités locales au passage. Chacun roule
à son rythme, les nuits se passent en camping.

Plus d'info sur le site : [cyclotourix.fr](https://cyclotourix.fr/).

## Le site

- `index.html` — page d'accueil
- `program.html` — détail des étapes
- `tour_de_gaule.html` — présentation du concept
- `contact.html` — formulaire d'inscription
- `css/style.css` — feuille de styles commune
- `assets/` — polices, images et icônes
